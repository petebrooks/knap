# Knap
Ruby CLI for knapsack problem

### Install
1. `gem install thor`
2. `git clone <URL>`
3. `cd knap`
4. `thor knap load <FILE_NAME>` || `thor knap help` || `thor knap test`

### Menu Generator
Includes CLI for generating test menus

1. `cd spec`
2. `thor menu_gen`

All test files are saved in spec/test_menus
